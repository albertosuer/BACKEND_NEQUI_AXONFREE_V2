## ⚠️ Errores Comunes

### ❌ Error 1: Usar el comando viejo
```
/nequiaxonfree 3001234567 0515 500000
```
**Solución:** Usa `/nequiaxonlabs` en lugar de `/nequiaxonfree`

### ❌ Error 2: Olvidar parámetros
```
/nequiaxonlabs 3001234567
```
**Solución:** Debes incluir los 3 parámetros: numero, pin y saldo

### ❌ Error 3: Formato incorrecto del saldo
```
/nequiaxonlabs 3001234567 0515 500.000
```
**Solución:** No uses puntos ni comas en el saldo, solo números: `500000`

### ❌ Error 4: PIN sin completar
```
/nequiaxonlabs 3001234567 15 500000
```
**Solución:** El PIN debe tener 4 dígitos: `0015` (agrega ceros al inicio si es necesario)
